# Room Mate

Room Mate is a Laravel MVC web application for managing shared house chores and expenses.

## Features
- User authentication
- Chore management
- Expense tracking
- Role-based access
- Dashboard

## Tech Stack
- Laravel
- PHP
- MySQL
- Blade
- GitHub

Room mate which is a web application designed to help people living in shared houses manage chores and shared expenses.because in shared living situations, disagreements can happen because chores and bills are not clearly tracked. This application would allow house members to add chores, mark them as completed, and record shared expenses such as groceries or utility bills. A house admin would also be able to manage members and assign chores. From a technical perspective, it fits the assignment well because we can implement *CRUD functionality for chores, use **user authentication and role-based authorisation, and store the data in a **relational database using Laravel migrations*. The application would include pages such as a dashboard, chore management, bill tracking, login/registration, and house settings. I think it’s a practical idea with a clear problem to solve, and it should allow us to demonstrate the required Laravel and MVC concepts effectively
